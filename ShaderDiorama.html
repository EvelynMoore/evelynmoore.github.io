Retrofuturistic Shader Diorama

A diorama created in Unity containing many different HLSL/CGPROGRAM shaders and many shadergraph shaders.

A stencil is used to mask out windows on different faces of a cube, and several tiny worlds are contained within.

The user can even enter these worlds, which is unusual for a stencil based approach.

I also wrote a raymarching based cloud shader to render a nebula effect.
