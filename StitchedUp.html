Stitched Up

Blender & Unity - C#
Design & Programming
Uni Assignment
Team of 10

Stitched Up was a final year university project that I contributed to during my second year. I was responsible for prototyping menu/gameplay transition animations and programming UI elements including the character selection screen. You can find the game here.

One of my tasks was the code implementation of the character select screen. There can be up to 4 players, and 4 characters to choose from.

The game was invited to the annual GAMEX showcase. It was great to see so many people enjoying the game together!

The game also won the COMX award for best trailer!
